# ISTIC UI - Começando

Bem-vindo ao ISTIC UI! Este guia vai te orientar no processo de download e uso da nossa biblioteca.

## Step 1: Instalação

Para usar o ISTIC UI no seu projeto, você precisa instalá-lo via pnpm, npm ou yarn. Abra seu terminal e execute um dos seguintes comandos:

```bash
pnpm install @istic-ui/react
```

```bash
npm install @istic-ui/react
```

```bash
yarn add @istic-ui/react
```
