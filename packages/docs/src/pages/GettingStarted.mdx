# STICK UI - Getting Started

Welcome to STICK UI! This guide will walk you through the process of downloading and using our library.

## Step 1: Installation

To use STICK UI in your project, you need to install it via pnpm, npm or yarn. Open your terminal and run one of the following commands:

```bash
pnpm install @stick-ui/lib
```

```bash
npm install @stick-ui/lib
```

```bash
yarn add @stick-ui/lib
```
